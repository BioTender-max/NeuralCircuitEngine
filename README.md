# NeuralCircuitEngine

Simulate neural circuit dynamics with excitatory/inhibitory neuron populations, interneuron subtypes, and firing statistics.

## Features

- Excitatory/inhibitory neuron population modeling (400/100 split)
- Interneuron subtype classification (PV, SST, VIP)
- Firing rate distribution and coefficient of variation analysis
- Connectome-based synaptic weight simulation
- Neural coding efficiency and population dynamics

## Results

500 neurons (400 Exc/100 Inh); PV=40, SST=35, VIP=25; Mean firing=19.55 Hz; CV=0.764

## Usage

```bash
pip install numpy scipy matplotlib
python neural_circuit_engine.py
```
